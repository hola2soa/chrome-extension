### Usage
Start development server. The server will run on localhost:4000
```
$ npm run dev
```

Build production files with Webpack
```
$ npm run build
```

Run ESlint
```
$ npm run lint
```

Run tests. The tests are run in NODE_ENV=test environment. You need to copy another config.
```
$ npm run test
```
